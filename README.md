# NILU iOSアプリ（非公式）

VALUのパロディサービス「[NILU](https://nilu-is.appspot.com/)」の非公式アプリです。

![](./screenshot.png)

## Usage

Constants.swift の `profileUrl` をご自身のプロフィールURLに書き換えてビルドしてください。

## Requirements

Swift 3.0 / iOS 10.0+
