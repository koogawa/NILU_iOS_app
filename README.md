# NILU iOSアプリ（非公式）

[NILU \| だれでも、株式会社のように、あなたの無をトレード。](https://nilu-is.appspot.com/) の非公式アプリです。

## Usage

Constants.swift の `profileUrl` を自分のプロフィールURLに書き換えてください。

## Requirements

Swift 3.0 / iOS 10.0+
